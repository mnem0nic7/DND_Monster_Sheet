AC Override Test

Test the AC override functionality:

  1. Visit http://localhost:4100
  2. Click "Create New Creature"
  3. Click on the "Basic Stats" section to edit it
  4. Check the "Override AC" checkbox
  5. The AC field should become editable (not grayed out)
  6. Try entering a custom AC value (e.g., 15, or even something high like 500)
  7. The field should accept the value and display it

What to check:

  • The AC field changes from read-only (grayed out) to editable when override is checked
  • You can type numbers into the AC field when override is checked
  • The label changes from "(Calculated)" to "(Manual)"
  • When you uncheck override, the field becomes read-only again and recalculates

Debug info:

Check the browser console (F12) for debug messages when toggling the override checkbox.
